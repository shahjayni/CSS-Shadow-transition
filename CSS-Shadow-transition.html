1. Create an hover effect

Retain the existing markup and css:

  1. make the image fits the div
  2. make the div a circle
  3. add a drop shadow to the circle, e.g. blur 30px, spread 15px, 20% black
  4. on hover (over the circle), create a zoom-in effect for the image. The size of the circle is unchanged (100px). The duration of the zoom-in effect should be 350ms.
